Show user tickets from HelpScout, ZenDesk, FreshDesk and Teamwork in wordpress. Users can create new support tickets and reply to old tickets.

== Description ==

Show user tickets from [HelpScout](https://www.helpscout.net/), [ZenDesk](https://www.zendesk.com/), [FreshDesk](https://freshdesk.com/) and [Teamwork Desk](https://www.teamworkdesk.com/) in wordpress. Users can create new support tickets and reply to old tickets from wordpress. Learn more at [crmperks.com](https://www.crmperks.com/plugins/support-plugins/support-x/?utm_source=wordpress&utm_medium=directory&utm_campaign=supportx+readme)

== Supported Ticket Systems ==

* [HelpScout](https://www.helpscout.net/)
* [ZenDesk](https://www.zendesk.com/)
* [FreshDesk](https://freshdesk.com/)
* [Teamwork Desk](https://www.teamworkdesk.com/)

== Key Features ==

* Display all tickets related to a user in table form.
* You can use shortcode to display a 'Create a Ticket' form and all tickets related to a user.
* You can prevent spams by enabling Google reCaptcha before submitting a ticket.
* Displays user tickets and 'Create a Ticket' form in "my-account" section of WooCommerce.
* Plugin works with WooCommerce and HelpScout. When viewing a ticket in helpScout, It displays all WooCommerce orders of that user in Help Scout.
